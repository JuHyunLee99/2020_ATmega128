# 2020_ATmega128
마이크로프로세서 ATmega128  

## Term Project 
### Dot Matrix에 동영상 표시하기
- Dot Matrix에 4가자ㅣ 이상의 연속동작 구성
- 버튼 #0을 누를 때마다 다음 동작을 표시
- 버튼 #0을 누르고 있으면 연속적으로 동작을 이동하여 동영상으로 표시
- 버튼 #1을 누르고 연속동작으로 동작 표시하고, #0를 누르면 구분동작 표시모드로 이동
- 현재 동작이름을 LCD에 표시
